## Setting Up Your Python Development Environment

Before diving into Django, you need to set up Python and essential tools like `pip` and virtual environments.

### 1. Install Python

First, check if Python is installed on your system:

```sh
python --version
```

If Python is not installed, download and install it from [python.org](https://www.python.org/downloads/).

### 2. Install the Python Package Manager (pip)

`pip` is the package manager for Python, used to install libraries and frameworks like Django. It usually comes pre-installed with Python. Check if you have it:

```sh
pip --version
```

If not, install it using:

```sh
python -m ensurepip --default-pip
```

### 3. Create a Virtual Environment

Virtual environments allow you to create isolated spaces for different projects, preventing dependency conflicts.

To create a virtual environment, run:

```sh
python -m venv myenv
```

Activate the virtual environment:

- **Windows:**

  ```sh
  myenv\Scripts\activate
  ```

- **macOS/Linux:**

  ```sh
  source myenv/bin/activate
  ```

To deactivate it, simply run:

```sh
deactivate
```

### 4. Install Django

Once your virtual environment is activated, install Django using `pip`:

```sh
pip install django
```

Verify the installation:

```sh
django-admin --version
```
